# Проектирование и реализация системы "Новостной агрегатор"
Реализовано с помощью laravel, native PHP

## Что в проекте
Парсинг RSS осуществляется командой ``php artisan app:import-feed``. В идеале её лучше поставить на ежеминутное выполнение в cron, но здесь для демонстрации она выполняется перед загрузкой страницы.

Полученные заголовки хранятся в базе mySQL в таблице feeds. 

На выходе - единая страница с новостными заголовками, полученными от РИА. По клику открывается оригинальная статья.

Страницы результата кэшируются, учитывается фильтрация результата. Кэш обновляется ежеминутно.

## Что требуется для развёртывания
Выполнить команды:

``composer install``

``npm install``

``php artisan key:generate``

``php artisan native:install``

``php artisan native:migrate``

``php artisan native:serve``
